PEREKAT: THE FLAPPY Y.O.B.A.
======
## ALLO YOBA ETO TI?
Игра была сделана за 2 дня на HTML5/JS и является первой из серии PEREKAT.
Она повествует о приключениях Желтого Баттхертного Кружка по миру столбов в поисках Еще Того Двача.
